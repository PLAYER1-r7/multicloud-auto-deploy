## 🎉 主な機能追加・改善

### 🔍 検索機能の統合

- タグ検索と本文検索を同一パネルに統合し、使いやすさを向上
- ヘッダの検索ボタンから両方の検索機能に一括アクセス可能

### 🎨 テーマ対応の大幅改善

- ライトモード/ダークモードでのテキスト視認性を大幅に向上
- CSS変数による完全なテーマ切り替えサポート
- 全コンポーネントでのテーマ一貫性を確保

### ✨ LiquidGlass透明感の強化

- 背景に5層の放射状グラデーションパターンを追加
- ドットパターン（1.5px、30pxグリッド）の実装
- ブラー効果を40pxに強化（saturate 220%）

### 📋 バージョン情報の表示改善

- ヘッダにバージョン番号を表示
- バージョン履歴モーダルで変更履歴を確認可能
- GitHubリリースページへのリンクを追加

### 🎯 技術的改善

- **Tailwind CSS v4** への移行（@tailwindcss/postcss）
- Lambda Layer実装による **99%のサイズ削減**
- セキュリティ強化とバリデーション改善
- コード品質の向上とリファクタリング
- 複数画像削除のサポート
- プリサイン済みURLの有効期限設定
