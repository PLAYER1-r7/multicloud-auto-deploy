# v1.2.3

**リリース日**: 2026年1月17日

## 主な変更点

### 🎨 惑星テーマの背景デザイン実装

ライトモード・ダークモードそれぞれに異なる惑星をテーマにした背景デザインを実装しました。

#### ライトモード

- 地球風の惑星デザイン
- 3層の雲レイヤー（楕円形のグラデーション）
- 太陽のレンズフレア効果を追加
- クラウドドリフトアニメーション（60秒周期）

#### ダークモード

- 木星/土星風のガスジャイアント惑星
- 36個以上の個別の星を配置
  - サイズ: 0.6px〜2px
  - カラー: 白、青みがかった白、暖色系
  - 透明度: 0.5〜1.0
- スターシマーアニメーション（100秒周期）

### 🔧 改善点

- ドットパターンを削除し、より自然な宇宙空間の表現に変更
- 繰り返しパターンから個別配置の星へ移行
- 回転アニメーションから自然現象のアニメーション（雲の移動、星の瞬き）へ変更

## 技術詳細

- Pure CSS gradients（画像不使用）
- 42層のbackground-layerを使用（ダークモード）
- CSS Keyframe Animations
- レスポンシブデザイン対応
