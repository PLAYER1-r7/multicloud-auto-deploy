# Docs Start Here

このファイルは `docs/` 内の入口です。

## Purpose

`docs/` は製品実装のための文書を管理します。

- アーキテクチャ
- API / CLI 仕様
- セットアップ
- 実装制約
- テスト/運用チェックリスト

この領域が答える問い:

- この製品をどう実装するか
- この製品をどう実行するか
- この製品をどう保守するか

## Reading Order

1. `docs/AI_AGENT_00_CRITICAL_RULES_JA.md`
2. `docs/AI_AGENT_01_CONTEXT_JA.md`
3. `docs/AI_AGENT_02_ARCHITECTURE_JA.md`
4. `docs/AI_AGENT_03_API_JA.md` または対象領域の専門文書

## Out Of Scope

GitHub 上の運用手順（Issue/PR/Projects/自動化運用）は `.github/` で管理します。

- 入口: `/.github/00_START_HERE.md`
